Welcome to the
Choate Zoom Coordinator

This website allows students and teachers to easily save and access Zoom meeting IDs. Meeting IDs are displayed on buttons and link directly to the meeting.

Developers' note: Teacher names are not in the correct order due to inconsistencies in the way ITS gave the name of each teacher. Some of the names were given in reverse-order, others were given in an incorrect reverse order, and others were given in a different, unspecified ordering scheme. Please be patient while we work to fix this.

Information for Teachers:

Teachers can set their office hours and classes' Zoom meeting IDs. Any changes made by a teacher will overwrite a student's individually set meeting ID.

We strongly recommend that teachers maintain a single Zoom ID per class throughout the term.

Information for Students:

Students can set their classes' Zoom meeting IDs when it has not been set by the teacher, but these changes do not affect other students' meeting IDs. In addition, students can view and search for a teacher's office hours ID when it is set.

Classes are ordered chronologically and display their starting times.
